This directory contains a simple example of using the olympus2tf converter to
produce models that can be used with TensorFlow.js. This example uses
[tfjs.converters.olympus_conversion.convert_olympus() API](https://github.com/tensorflow/tfjs/tree/master/tfjs-converter#calling-a-converter-function-in-python-flaxolympus),
which is part of TensorFlow.JS, and uses olympus2tf under the hood.

See the following blog post for more details:
[OLYMPUS on the Web with TensorFlow.js](https://blog.tensorflow.org/2022/08/olympus-on-web-with-tensorflowjs.html).
